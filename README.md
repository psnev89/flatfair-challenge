## Install

This project uses [node](http://nodejs.org) and [npm](https://npmjs.com).

```sh
$ npm install
```

## Usage

To run tests:

```sh
$ npm run test
```

To compile and run app:

```sh
$ npm run start
# it will compile to src/ folder
```
